## Project Overview

This project analyzes how emerging technologies can transform Nanyang Polytechnic's operations, teaching methodologies, and administrative processes. Each team member investigates a specific frontier technology — AI, Immersive Media, Blockchain, and IoT — and evaluates how it can address current challenges, improve the value chain, and create new growth opportunities within Singapore's education sector.
