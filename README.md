# IT2392 Emerging Technology Screening — Team Proposal

## Emerging Technologies for Nanyang Polytechnic (Education Sector)

This repository contains the team proposal for the IT2392 Emerging Technology Screening assignment, analysing how emerging technologies can enhance operations and educational outcomes at Nanyang Polytechnic (NYP) in Singapore.

### Documents

- **[proposal.md](proposal.md)** — Full team proposal document covering:
  - Organisation overview and current challenges
  - Market size and revenue potential analysis
  - Four emerging technology analyses (AI, Immersive Media, Cybersecurity, IoT)
  - Cost-benefit analysis for each technology
  - Impact on digital economy and future markets
  - References with URLs

### Key Topics Covered

1. **Artificial Intelligence (AI)** — Adaptive learning, predictive analytics, AI-assisted assessment, administrative automation
2. **Immersive Media (AR/VR)** — Virtual labs, AR-enhanced learning, virtual field trips, immersive assessment
3. **Cybersecurity** — Zero-Trust Architecture, Security Operations Centre, cybersecurity awareness, data protection
4. **Internet of Things (IoT)** — Smart building management, space utilisation, predictive maintenance, environmental monitoring
